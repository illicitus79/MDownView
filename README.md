# MdownView

MdownView is a lightweight Markdown editor and viewer for macOS. It supports tabbed editing, split view, and a polished light/dark theme.

## Features
- Edit, preview, or split view for Markdown.
- Multiple files open in tabs.
- Tabs docked on top or on the left.
- Light, dark, or system appearance.
- Open and save Markdown files (`.md`, `.markdown`, plain text).
- Word, line, and character counts.

## Shortcuts
- New tab: `⌘N`
- Open files: `⌘O`
- Save current tab: `⌘S`
- Save all tabs: `⌘⌥S`
- Save As: `⇧⌘S`

## Notes
- Save All only saves tabs that already have a file location. Use Save As to save new tabs.

## Build
Open `MdownView/MdownView.xcodeproj` in Xcode and run the app.
